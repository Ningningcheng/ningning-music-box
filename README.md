# 宁宁音乐盒

**宁宁音乐盒**是一款用于无名杀的内置音乐播放器扩展。

扩展可以读取自身 `music` 文件夹内的本地音频文件，并在对局界面顶部显示一个半透明悬浮播放器，用于在游戏过程中播放自定义音乐。

- 扩展名称：宁宁音乐盒
- 当前版本：V0.9.0
- 作者：宁宁澄

---

## 主要功能

### 基础播放

- 播放、暂停和停止音乐
- 上一首、下一首
- 播放进度显示与跳转
- 独立音量调节
- 播放速度调节
- 播放失败时自动跳过

目前支持的播放速度：
- ×0.5
- ×0.75
- ×1.0
- ×1.25
- ×1.5
- ×2.0

### 播放模式

目前支持以下播放模式：
- 顺序播放
- 列表循环
- 随机播放
- 单曲循环
- 收藏播放

### 播放列表

- 查看当前已扫描到的全部音乐
- 点击歌曲名称直接播放
- 当前播放歌曲高亮
- 收藏或取消收藏歌曲
- 修改歌曲在音乐盒内的显示名称
- 收藏状态与显示名称会保存到扩展配置中

修改显示名称不会修改真实磁盘文件名，也不会影响歌曲播放、收藏、歌词匹配与恢复播放。

### 悬浮窗

- 对局顶部显示半透明音乐播放器
- 支持锁定悬浮窗
- 支持自动收起
- 支持进入对局时默认收起
- 支持拖动并保存悬浮窗位置
- 支持本局隐藏和重新打开
- 收起后仍可使用播放列表、音量和拖动按钮
- 打开无名杀菜单或部分高层界面时自动隐藏，避免遮挡操作

### 歌词显示

歌词读取顺序如下：
1. 优先读取与歌曲同名的 `.lrc` 文件
2. 没有 `.lrc` 时，尝试读取音频文件内部的同步歌词
3. 没有可同步歌词时，显示短暂提示后自动隐藏歌词区域

只有带时间轴的歌词才能跟随播放进度显示。
普通整段歌词由于没有对应时间，不会进行同步显示。

### 恢复播放

可以在扩展设置中选择恢复播放方式：
- 重启与退出
- 仅重启
- 关闭

开启恢复后，可以记录：
- 当前歌曲
- 当前播放进度
- 播放或暂停状态

还可以单独关闭“恢复后继续进度”，使恢复歌曲时从头播放。

部分移动端 WebView 可能限制冷启动自动播放。如果歌曲和进度已经恢复，但没有立刻播放，进入游戏后点击一次界面即可继续。

### 与无名杀本体音乐的配合

播放音乐盒歌曲时，会自动暂停无名杀本体背景音乐。

暂停或停止音乐盒后，会恢复本体背景音乐。

切换到其他应用、回到桌面或电脑端切换窗口时，也可以根据设置自动暂停音乐盒，返回游戏后再继续播放。

## 支持的文件格式

当前支持的音频格式：
- `.mp3`
- `.ogg`
- `.wav`
- `.m4a`
- `.flac`

歌词文件格式：

- `.lrc`

扩展会读取音频格式内部保存的同步歌词。

## 扩展目录

```
宁宁音乐盒/
├─ extension.js
├─ info.json
├─ README.md
├─ NNC.jpg
├─ nncAvatar.js
├─ nncLyrics.js
├─ music/
│  ├─ 示例音乐.mp3
│  ├─ 示例音乐.lrc
│  ├─ 另一首音乐.flac
│  └─ ...
└─ styles/
   └─ musicBox.css
```

extension.js
音乐盒主程序，包含播放控制、悬浮窗、播放列表、导入、收藏和恢复播放等功能。

info.json
扩展基础信息。

README.md
扩展说明文档。

NNC.jpg
作者头像的本地备用图片。

nncAvatar.js
作者联网头像加载组件。网络头像无法读取时继续使用 NNC.jpg。

nncLyrics.js
歌词读取和解析组件，负责解析 LRC 与音频内嵌歌词。

music/
存放音乐与外置 LRC 歌词文件。

styles/musicBox.css
音乐盒悬浮窗与播放列表的样式文件。
